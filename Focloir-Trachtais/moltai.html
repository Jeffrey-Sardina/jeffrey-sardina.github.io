Moltaí faoi Théarmaí

Cé gur foclóir é seo a rinne mé do mo thráchtas féin, táim lán-sásta le moltaí faoi théarmaí nua, le ceartúcháin, agus eile. Más suim leat moladh a bhaineann le téarma(í) a chur isteach, déan Pull Request ar an leathanach GitHub seo: https://github.com/Jeffrey-Sardina/Focl-ir-Tr-chtais

Tá lán-fháilte romhat r-phost a sheoladh chugam chomh maith ag jeffreysardina [ag] duck [punc] com. (Muna bhfuil táithí agat le r-phoist @duck.com, is córas r-phost é a chosnaíonns tú ó spam agus ó ionsaithe ar do phríobháid. Tá tuilleadh eolais anseo: https://duckduckgo.com/email/settings/autofill)

Moltaí faoin Suíomh

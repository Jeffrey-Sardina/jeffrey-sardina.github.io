pairwise function | feidhm de phéire (pointí)

pairwise function (ainmfhocal): feidhm de phéire (pointí)
sainmhíniú (ga): Feidhm a bhfuil péire pointí sonraí mar ionchur aici (chun iad a chur i gcomparáid lena chéile, mar shampla).
sainmhíniú (en): A function that takes a pair of data points as input (for example, to compare them).

tagairtí:
  • Féach ar an téarma 'feidhm / function'.
  • péire: De Bhaldraithe (1978), Dineen (1934), Ó Dónaill et al. (1991), Ó Dónaill (1977)
  • Féach ar an téarma 'pointwise function / feidhm de phointe'.
